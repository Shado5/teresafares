# Portfolio Website

This repository contains the source code for my personal portfolio website, hosted using GitHub Pages.

The website showcases my projects, skills and background in IT.

## Live Website

https://teresafares.com/

## Why This Repository Is Public

This repository is public because GitHub Pages requires public repositories on free GitHub accounts in order to host static websites.  
The code is intentionally public for transparency and demonstration purposes.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript   
- GitHub Pages
